# Flag Quest

A browser-based 2D platformer game where you play as a mushroom guy navigating platforms to collect coins and reach the flag.

## How to Play

Open `index.html` in your browser — no build step or server required.

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move left / right |
| Space / W / Up Arrow | Jump |

### Objective

- Collect all coins on the level
- Reach the flag at the end to win
- Use a key to continue if you get stuck

## Features

- Responsive canvas that scales to any screen size
- Moving platforms
- Coin collection counter
- Key / continue system

## Running Locally

Just open the file:

```bash
open index.html
```

Or drag `index.html` into your browser.
